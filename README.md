# Projeto de E-commerce para o Projeto Integrador do Quarto Semestre

# Sobre o projeto

O projeto foi feito para cumprir os requisitos do Projeto Integrador, onde deveriamos criar um E-commerce, que nesse caso, foi de tênis, que tivesse tanto a parte completa do usuário, ou seja, da criação do cliente até o pagamento do pedido e verificação do pedio, e também a parte do administrador e estoquista, onde cada um tinha suas partes separadas no sistema, um para editar informações, e outro para acompanhar os pedidos.

A aplicação consiste em um E-commerce que o cliente pode criar seu usuário, editar suas informações, realizar sua compra, e acompanhar seu pedido.

# Tecnologias utilizadas

- Spring Boot
## Front end

- CSS
- JavaScript
